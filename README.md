# Running the example

```bash
$ mkdir -p ~/dev
$ cd ~/dev
$ git clone https://github.com/telecoms-intelligence/netflix-study.git
$ mkdir -p data
$ cd data
$ wget wget https://archive.org/download/nf_prize_dataset.tar/nf_prize_dataset.tar.gz
$ tar zxf nf_prize_dataset.tar.gz

```
